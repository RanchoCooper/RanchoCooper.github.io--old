前言

花了一个下午多的时间看完了Intermediate Python 的中文译本, 在此稍作整理和总结. 按照个人的理解重新分门别类, 同时会加入辅助例程和其他深入探究, 细节上有任何新发现都会直接加进来(不保证行文的流畅..), 任何问题欢迎留言讨论.

感谢中文译本的翻译人员

本文虽然名为进阶, 但完全适合仅了解基本语法的Green Man, 部分特性会结合3.5的源码加以深入, 刨根问底. 通篇文章很长, 但绝对值得花上几个小时细细一读, 你要明白, 阅读任何干货, 无思考, 不吸收.

总的来讲, Python高级语法无外乎生成器, 装饰器, 上下文管理, 以及一些函数式编程的特性, 以及数据结构和性能优化方面的东西
